# AlertFlow - Real-Time Notification Hub

**Internship ID:** CTTS114

AlertFlow is a modern full-stack Real-Time Notification Management Platform designed to create, monitor, and manage notifications efficiently through an intuitive dashboard interface.

The application provides centralized notification tracking, priority alert management, activity monitoring, and real-time notification workflows suitable for enterprise and productivity environments.

---

## 🚀 Live Demo

https://alertflow-jtk3.onrender.com

## ✨ Features

* Create Real-Time Notifications
* Notification Feed Dashboard
* Priority Alert Management
* Read / Unread Status Tracking
* Notification Search
* Category Filtering
* Activity Timeline
* Alert Analytics Dashboard
* Responsive Design
* Modern SaaS Interface
